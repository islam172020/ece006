The history of computer architecture generations

Generation technology software Example
First (1945-54) Vacuum tubes and relay memories Machine language (i.e., 1’s and 0’s) ENIAC
Second (1955-64) Transistors FORTRAN IBM 7090
Third (1965-74) MSI (medium-scale integration); pipelining; cache Time-sharing, parallel processing PDP-8, IBM 370
fourth (1975-90) VLSI (very large scale integration), vectorisation , multicomputers Multiprocessor OS; compiler environments for parallel processing Sun SPARC, Cray X-MP
Fifth (1991—now) ULSI (ultra-large scale integration) Massively parallel processing Cray MPP,CM2
